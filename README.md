# 🎮 ASP.NET Core Web API Practice

This repository contains my practice projects and exercises using **ASP.NET Core Web API**.

## 🚀 About This Project

This project is built to practice and understand:

- ASP.NET Core Web API fundamentals
- Controllers and Routing
- HTTP Methods (GET, POST, PUT, DELETE)
- Model Binding
- RESTful API Design
- Testing APIs using Swagger / Scalar
- In-memory data storage

Currently, the API includes a sample **Video Game API** that allows retrieving video game data.

## 🛠 Technologies Used

- ASP.NET Core
- C#
- .NET 8
- Swagger / Scalar for API testing
